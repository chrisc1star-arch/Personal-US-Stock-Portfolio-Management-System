# Personal-US-Stock-Portfolio-Management-System

This repository contains the course project submission for **COMP8090SEF Data Structures and Algorithms**. The project is designed to solve real-life financial tracking problems and is divided into two main components.

## Repository Structure

### 1. Task 1: OOP-Based Application
A multi-module Python application for managing a personal US stock portfolio. 
* **Objective:** Track stock positions, calculate real-time market values, and manage account balances.
* **Core Concepts:** Strictly implements Object-Oriented Programming (OOP) principles, including **Encapsulation** (protecting balance/quantity data), **Inheritance** (deriving specific assets from a base class), and **Polymorphism** (overriding valuation methods).
* **Modules:** Structured into `main.py`, `models.py`, and `utils.py`.

### 2. Task 2: Data Structure & Algorithm Self-Study
An independent exploration of a data structure and an algorithm not covered in the standard curriculum, applied to financial data analytics.
* **Data Structure:** Max-Heap (used to efficiently retrieve the top-performing stocks without sorting the entire dataset).
* **Algorithm:** Heap Sort (O(n log n) sorting algorithm applied to historical stock returns).

## How to Run
Please navigate to the respective subdirectories (`Task1_Portfolio/` or `Task2_DSA/`) for specific source code files and further instructions.
